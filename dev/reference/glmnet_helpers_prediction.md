# Organize glmnet predictions

This function is for developer use and organizes predictions from glmnet
models.

## Usage

``` r
.organize_glmnet_pred(x, object)
```

## Arguments

- x:

  Predictions as returned by the
  [`predict()`](https://rdrr.io/r/stats/predict.html) method for glmnet
  models.

- object:

  An object of class `model_fit`.

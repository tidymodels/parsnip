# Wrapper for `qrnn::mcqrnn.fit()`

Wrapper for `qrnn::mcqrnn.fit()`

## Usage

``` r
mcqrnn_train(x, y, ..., trace, tau)
```

## Arguments

- x:

  Predictor matrix

- y:

  outcome vector

- ...:

  Options to pass to `qrnn::mcqrnn.fit()`

- trace:

  Logical for printing

- tau:

  Vector of quantile levels.

# Determine required packages for a model

**\[deprecated\]**

## Usage

``` r
req_pkgs(x, ...)
```

## Arguments

- x:

  A [model
  specification](https://parsnip.tidymodels.org/dev/reference/model_spec.md)
  or [fit](https://parsnip.tidymodels.org/dev/reference/model_fit.md).

- ...:

  Not used.

## Value

A character string of package names (if any).

## Details

This function has been deprecated in favor of
[`required_pkgs()`](https://generics.r-lib.org/reference/required_pkgs.html).
